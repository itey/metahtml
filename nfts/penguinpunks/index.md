# PenguinPunks

一款类似于 Club Penguin（但使用区块链）的游戏，用户可以使用他们的 PenguinPunk 与其他 PenguinPunk 所有者聊天、漫游、放松和玩耍，它正在开发中（可能）。

他们将努力使这成为现实。

每个 PenguinPunk 持有者都可以查看和下载他们的企鹅的 3D 版本。这个 3D 版本是一个 GIF，它是他们企鹅的 8 个不同角度的组合。
